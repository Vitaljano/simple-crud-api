## Getting Started
- clone repository to your local environment and install all dependencies ```npm i```
- create in project file <code>.env</code> and write port ```PORT=8080```
- server starts on ```http://localhost:8080```
- to run project in dev mode ```npm run start:dev```
- to run project in dev mode ```npm run start:build```
- run test ```npm run test```
