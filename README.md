## Getting Started
- clone repository to your local environment and install all dependencies <code>npm i</code>
- create in project file <code>.env</code> and write port <code>PORT=8080</codes>
- server starts on <codes>http://localhost:8080</codes>
- to run project in dev mode <code>npm run start:dev</code>
- to run project in dev mode <code>npm run start:build</code>
- run test <code>npm run test</code>
